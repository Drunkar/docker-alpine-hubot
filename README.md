# Drunkar/hubot-alpine


## Build Info

- Alpine Linux: 3.9
- node: 10.15.3
- yarn: 1.13.0

## Dependencies

- docker
- docker-compose

## Default Scripts

Please see `external-scripts.json`.


## Usage

After editing `.env`, `hubot.conf` and `external-scripts.json` to meet your needs,

```
docker-compose up -d
```
